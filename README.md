# CoboGen-UR5

Cognitive control system for a UR5 collaborative robot using generative AI and natural language commands.

## Overview

This project implements a cognitive architecture that allows controlling a UR5 robot through natural language (text or voice).

## Architecture

User Input → AI → Command Parser → Robot Controller → UR5

## Features

- Natural language control (Spanish)
- Voice and text input
- AI-based command interpretation
- UR5 control via URScript / URX

## Example Commands

- "Ve a la posición inicial"
- "Coge la pieza y colócala en la caja"
- "Detente"

## Tech Stack

- Python
- URScript / URX
- Generative AI
